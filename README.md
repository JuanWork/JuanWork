# 🎵 Music-Control (Universal Flow)

> **"Humanizando o desenvolvimento. Sinfonia e paz no código."**

Controle de áudio nativo para Linux (Rhythmbox) construído com **Node.js** e **AOP (Aspect-Oriented Programming)**. Um projeto que une a resiliência das estradas com a precisão do terminal.

## 🚀 Arquitetura & Infraestrutura
- **Hardware:** Intel i3 | NVIDIA GT 620 (Aceleração WebGL Ativa)
- **Storage Híbrido:** Código no **SSD** e Memória de Longa Duração (Logs/DB) no **HD de 1TB**.
- **Segurança:** Blindado com **2FA (Authy)** e **Tokens Granulares**.
- **Ambiente:** Linux Mint + **VimPank** (The Master Tool).

## 🛠️ Como rodar a "Trinha"
```bash
npm install
npm test
